# -- Project Summary

-   First, I understood the project requirements and identified that the
    main objective was to analyze airline route profitability using SQL
    and Excel/Power BI.
-   I imported the **AirlineRoutesData.csv** dataset and verified the
    data by checking data types, removing duplicates (if any), and
    ensuring all required columns were available.
-   I performed SQL analysis to calculate route-wise revenue,
    operational cost, profit, seat occupancy, monthly profit trends, and
    route rankings.
-   The SQL query results were exported and used to build an interactive
    dashboard in Excel/Power BI.
-   I created KPI cards to display key business metrics such as Total
    Revenue, Total Profit, Total Flights, Occupancy Rate, and other
    performance indicators.
-   Various visualizations such as bar charts, line charts, donut
    charts, and stacked column charts were created to analyze route
    performance and profitability.
-   Slicers and filters for **Aircraft Type**, **Flight Month**, and
    **Route Code** were added to make the dashboard interactive.
-   Finally, the dashboard was used to identify profitable routes,
    loss-making routes, occupancy trends, and monthly business
    performance, helping management make better operational and
    financial decisions.
